基于 https://talktao.github.io/Vue3-Vite-Vant-TS-H5/#/home
前端vue3的开发模板

## 安装依赖

```bash
npm install

需要本地启动dify，连接本地服务http://localhost/v1 
config/assistantConfig.js
- 对应配置各个agent的不同APIKey及说明信息
- 同时配置科大讯飞的TTS的秘钥信息
```

## 运行

```bash
npm run dev
```

## 打包

```bash
npm run build
```
